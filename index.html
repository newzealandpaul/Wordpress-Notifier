About

Do you leave your Wordpress blog's comment admin page open in Safari all day long, clicking refresh every two minutes?

Are you sick of your inbox being clogged up with Wordpress "new comment" emails?

If so, then this application is for you!

Wordpress Notifier is a Open Source (GPL) application that alerts you when new unapproved comments are posted on your Wordpress blog.

It lives in the Mac OS X Status Bar and displays the current unapproved comment count.

Wordpress Notifier will work with Wordpress version 2.7 or later and with Wordpress.com.

Features

  • Current
    • Free and Open Source
    • Works with Wordpress 2.7 or later.
    • Works with Wordpress.com
    • Displays comment count in the Status Bar.
    • Displays comments in Status Bar menu.
    • Growl Integration.
