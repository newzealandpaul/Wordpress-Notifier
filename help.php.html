Help / Troubleshooting

First steps

If you are having problems these are the first three things you should check:

1) Make sure your blog is running Wordpress 2.7 or later.

2) Make sure XML-RPC is enabled (These instructions explain how to enable it).

3) Ensure that the preferences are setup correctly, especially the URL.

Explanation of error messages

Wordpress URL is incorrect. HTTP 404 page not found error.

The URL you set in your preferences is not correct. Make sure you are pointing to the directory that holds you blog and include a trailing slash "/".

These examples are incorrect:

  • http://www.myblog.com/blog/index.php (index.php not directory)
  • http://www.myblog.com/blog (does not include "/")

This is correct:

  • http://www.myblog.com/blog/

Unknown Error: Unable to parse XML response or Unknown XML-RPC error.

This could be a bug. See Bug Reporting.

Preferences not set.

You must setup your preferences in the preferences window.

Wordpress XML-RPC support has not enabled (See Help).

These instructions explain how to enable it.

FAQ

When will Mac OS X 10.4 be supported?

It won't, sorry, unless someone pays me enough to take the time to make it work ;)

When will Wordpress 2.6 be supported?

Short Answer: Wordpress 2.6 does not have the required comment API so it cannot work with Wordpress Notifier.

Long Answer: I have backported the comment API to Wordpress 2.6. Use at your own risk. Unzip the download and read the included instructions on how to install it.

When will Wordpress.com / Wordpress MU be supported?

Wordpress.com is supported right now! Wordpress MU lags behind Wordpress releases. I expect within a few months it will work with Wordpress Notifier.
