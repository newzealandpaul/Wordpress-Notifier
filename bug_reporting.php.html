Reporting bugs

Bug reports need to be detailed and must contain the following information:

1) Version of Wordpress Notifier (This can be found in the About window).

2) Mac OS X version. For example "10.5.4".

3) Type of Mac. For example "G4 iBook" or "Mac Mini".

4) Version of Wordpress blog software.

5) The Wordpress Blog URL (copy and paste exactly from the preferences dialog).

6) A list of all Wordpress plugins you have installed.

As well as the above, the output of the Console application (bundled with Mac OS X) needs to be provided. Console can be found at

/Application/Utilities/Console

Once Console is launched make sure "All Messages" is selected (It is at the top of the left hand side bar). The scroll down and copy/paste the most recent output from Wordpress Notifier.

It will look something like this:

29/11/08 • Nov  29 • 2:20:33 PM Wordpress Notifier[47567] Wordpress Notifier Error: 'Wordpress version unsupported. Please upgrade to Wordpress 2.7.' : -32601

Please file the bug report at GitHub.
